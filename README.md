# 🏥 Clinic Management System (Java)

## 📌 Overview
A console-based Clinic Management System developed using Java.  
The system allows managing patient records with persistent file storage.

---

## 🚀 Features
- Add Patient
- Search Patient by ID
- Delete Patient
- Clear All Patients
- Count Total Patients
- Duplicate ID Prevention
- Age Validation
- File Handling (Data Persistence)

---

## 🛠 Technologies Used
- Java
- OOP Concepts (Encapsulation, Modular Methods)
- ArrayList
- FileWriter & BufferedReader

---

## ▶ How to Run

1. Compile the program:
   javac ClinicApp.java

2. Run the program:
   java ClinicApp
   
---

## 💡 Key Concepts Applied
- Object-Oriented Programming
- File Handling for Persistent Storage
- Data Validation
- Modular Code Structure
